    <h1>Type  network_user </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type represents a user, however it is greatly reduced when compared to the full <a href="/docs/types/user">User</a> type to 
    reduce the amount of work that needs to be done to fetch it from multiple sites in the network.</p>

</div>
