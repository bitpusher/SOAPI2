    <h1>Type  user </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type describes a user on a Stack Exchange site.</p>

    <p>There are a number of different user types returned by the Stack Exchange API, depending on the method.  Others include <a href="/docs/types/shallow-user">shallow_user</a> and <a href="/docs/types/network-user">network_user</a>.</p>

</div>
