<div class="subheader">
    <h1>Type  filter </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type describes a filter on the Stack Exchange API.</p>

    <p>When passing a filter to methods in the API, it should be referred to by name alone.</p>

</div>

<h2>Fields</h2>
<div class="indented">
    
    
<div class="method">
    <div class="method-name">
        <span class="included" title="this field is included in the default filter"></span> filter 
    </div>
    <div class="method-description">
        string
        
        
    </div>
    <br style="clear: both;" />
</div>
<div class="method">
    <div class="method-name">
        <span class="included" title="this field is included in the default filter"></span> filter_type 
    </div>
    <div class="method-description">
        one of <code>safe</code>, <code>unsafe</code>, or <code>invalid</code>
        
        
    </div>
    <br style="clear: both;" />
</div>
<div class="method">
    <div class="method-name">
        <span class="included" title="this field is included in the default filter"></span> included_fields 
    </div>
    <div class="method-description">
        an array of strings
        
        
    </div>
    <br style="clear: both;" />
</div>

</div>

<div class="indented">
    <p>
        Fields marked with <span class="included"></span> are included in the default filter, those marked with 
        <span class="excluded"></span> are excluded in the default filter.
    </p>
</div>
