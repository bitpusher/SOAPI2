    <h1>Type  badge </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type represents a badge on a Stack Exchange site.</p>

    <p>Some badges, like Autobiographer, are held in common across all Stack Exchange sites.  Others, like most tag badges, vary
    on a site by site basis.</p>

    <p>Remember that ids are <i>never</i> guaranteed to be the same between sites, even if a badge exists on both sites.</p>

</div>
