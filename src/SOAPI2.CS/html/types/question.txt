<h2>Discussion</h2>
<div class="indented">
    
    <p>This type represents a question on one of the Stack Exchange sites, such as this famous <a href="http://stackoverflow.com/questions/1732348/regex-match-open-tags-except-xhtml-self-contained-tags">RegEx question</a>.</p>

    <p>This type is heavily inspired by the question page itself, and can optionally return comments and answers accordingly.</p>

</div>
