<div class="subheader">
    <h1>Type  event </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type describes an event that has recently occurred on a Stack Exchange site.</p>

    <p>A minimal ammount of information is present in these events for scaling purposes.  It is expected
    that most applications will make follow up calls to the API to "flesh out" the event objects for their
    own purposes.</p>

</div>

<h2>Fields</h2>
<div class="indented">
    
    
<div class="method">
    <div class="method-name">
        <span class="included" title="this field is included in the default filter"></span> creation_date 
    </div>
    <div class="method-description">
        <a href="/docs/dates">date</a>
        
        
    </div>
    <br style="clear: both;" />
</div>
<div class="method">
    <div class="method-name">
        <span class="included" title="this field is included in the default filter"></span> event_id 
    </div>
    <div class="method-description">
        the id of the object (answer, comment, question, or user) the event describes
        
        
    </div>
    <br style="clear: both;" />
</div>
<div class="method">
    <div class="method-name">
        <span class="included" title="this field is included in the default filter"></span> event_type 
    </div>
    <div class="method-description">
        one of <code>question_posted</code>, <code>answer_posted</code>, <code>comment_posted</code>, <code>post_edited</code>, or <code>user_created</code>
        
        
    </div>
    <br style="clear: both;" />
</div>
<div class="method">
    <div class="method-name">
        <span class="excluded" title="this field is excluded in the default filter"></span> excerpt 
    </div>
    <div class="method-description">
        string
        <br />unchanged in <a href="/docs/filters">unsafe filters</a>
        
    </div>
    <br style="clear: both;" />
</div>
<div class="method">
    <div class="method-name">
        <span class="excluded" title="this field is excluded in the default filter"></span> link 
    </div>
    <div class="method-description">
        string
        <br />unchanged in <a href="/docs/filters">unsafe filters</a>
        
    </div>
    <br style="clear: both;" />
</div>

</div>

    <p>
        Fields marked with <span class="included"></span> are included in the default filter, those marked with 
        <span class="excluded"></span> are excluded in the default filter.
    </p>
</div>
