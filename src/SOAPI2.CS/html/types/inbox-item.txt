<div class="subheader">
    <h1>Type  inbox_item </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type represents an item in a user's <a href="http://blog.stackoverflow.com/2010/09/new-global-inbox/">Global Inbox</a>.</p>

    <p>Be aware that the types of items returned by this method are subject to change at any time.  In particular, new types may be introduced
    without warning.  Applications should deal with these changes gracefully.</p>

    <p>Applications should not publish a user's inbox without their explicit consent, as while most item types are public in nature there are a few
    which are (and should remain) private.</p>

</div>
