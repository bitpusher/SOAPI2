<div class="subheader">
    <h1>Type  reputation </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type represents a change in reputation for a <a href="/docs/types/user">User</a>.</p>

    <p>All methods that return this data will scrub it to a degree, to increase the difficulty of correlating reputation changes
    with down voting.</p>

</div>
