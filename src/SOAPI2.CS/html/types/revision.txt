<div class="subheader">
    <h1>Type  revision </h1>
</div>

<h2>Discussion</h2>
<div class="indented">
    
    <p>This type represents that state of a <a href="/docs/types/post">Post</a> at some point in its history.</p>

    <p>Note that under some circumstances multiple edits can result in only a single revision.</p>

</div>
